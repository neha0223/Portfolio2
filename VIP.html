about the project

Sponsored by BMW

The Electronic ARTrium is a multimedia video game exhibit that detects a players' entire body to play and win the game. It combines life-sized digital imagery, animatronics, and sounds to create a transmedia experience.

Check out the instagram page below for more information.

My role consisted of sketching and modeling intricate structures for 3D environmental assets in Blender, and deploying those models into video game scenes using the Unity game engine for an interactive life-sized video game exhibit. I also facilitated lip sync animation sequences for mechatronics, digitally producing expected mouth movement using characters‚Äô auditory narrations within Unity's animator controller through C# interpolation scripts and Rhubarb's lip sync software.

I also assisted in system integration for interactive video game exhibit using Unity Multithreading Server through C# scripting in order to create a multimedia experience combining sound, mechatronics, and video. I continued the 3D Modeling of animatronic characters in Blender, creating digital renditions of expected robotic movements through rigging and animation of 3D models to be imported into Unity.

A smaller group, which I took part in, presented project outcomes to BMW representatives, receiving positive feedback and suggestions for future enhancements.
